# WeatharrStation

WeatharrStation is a small, self-hosted **TV-style** weather channel. It pulls local weather data from **NOAA/NWS**, turns it into polished graphics, and continuously plays them as a loop—just like a cable weather channel.

## What’s shown
- Current conditions
- Radar
- Hourly temperature curve
- 7-day forecast
- Regional map
- Text forecast
- Latest observations

## Ticker
A bottom ticker runs as **one continuous line** in this order:
**NOAA alerts → RSS headlines → NOAA alerts**.
